Install School ERP for quicker access

Add this app to your device for a full-screen experience and faster reopening.

On iPhone or iPad: tap Share, then Add to Home Screen.

In Safari on Mac: use File → Add to Dock (macOS Sonoma or later), or bookmark this page.

Firefox may not offer a full install for this app. Use a bookmark, or on Android Firefox try Add to Home screen from the menu when available.

Look for the install icon in the address bar or open the browser menu and choose Install School ERP / Install app.

Use your browser’s menu to install or add this site to your home screen, if your browser supports it.
